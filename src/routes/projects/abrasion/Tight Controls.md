## Details
The game will feature proven platform game movement controls included in games like Celeste, these features include:
	- [x] Increase velocity on turn
	- [x] Variable jump height
	- [x] Jump buffering
	- [x] Coyote time
	- [x] Clamped fall speed
	- [x] Edge detection
	- [x] Apex modifiers
